## Features Covered
- GET all products
- GET single product by ID
- POST create new product
- PUT update product
- DELETE remove product
- Negative cases (invalid ID, missing fields,Wrong Data Type)

## How to Use
1. Import the collection JSON file into Postman
2. Run individual requests or the full collection
3. Check status codes (200 OK, 404 Not Found, etc.) and response data
